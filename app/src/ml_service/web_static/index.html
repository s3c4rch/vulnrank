vulnrank

Интерфейс приоритизации security findings

ML-based triage

От raw findings к понятному приоритету без ручного хаоса

`vulnrank` принимает результаты security-анализа, ставит им приоритет и хранит историю обработки. Веб-кабинет работает поверх уже существующего REST API и очереди RabbitMQ.

Что делает сервис Priority triage

Помогает быстрее выделить findings, которые стоит разобрать первыми.

Пример входа JSON / CSV scan output

Scanner, severity, CVSS, asset type, port, CVE signal и краткое описание.

Роли User / Admin

User отправляет задачи и заявки на пополнение. Admin модерирует заявки и видит системный обзор.
